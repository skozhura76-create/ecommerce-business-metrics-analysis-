# ecommerce-# E-commerce Business Metrics Analysis

## 📌 Project Overview
This project focuses on analyzing transactional E-commerce data to evaluate key business performance metrics. The goal was to transform raw data into actionable business insights using SQL and Google Sheets.

## 🛠 Tools & Technologies
- SQL
- Google Sheets
- Data Cleaning
- KPI Analysis
- Dashboard Development

## 🔍 Project Steps
- Cleaned and structured raw data (duplicates, missing values, data types)
- Performed exploratory data analysis (EDA)
- Aggregated sales data by month
- Wrote SQL queries using JOIN, GROUP BY, SUM, COUNT, AVG
- Calculated key KPIs (Revenue, AOV, Total Users)
- Built an interactive dashboard

## 📊 Key Metrics
- Total Revenue
- Average Order Value (AOV)
- Monthly Sales Trend
- Largest Order Value

## 💡 Business Insights
- Identified revenue concentration patterns
- Detected seasonal sales fluctuations
- Highlighted high-performing product categories
- ## 🚀 Outcome
Delivered a structured dashboard and business recommendations to support data-driven decision-making.
